Les alias cools de bibi

Télécharger par lots

J'utilise cet alias pour récupérer des fichier par lots sur un serveur http. En général, c'est pour récupérer le contenu des torrents téléchargés à l'aide de ma seedbox (boooouuh)

alias dl='wget  --user [user] --password [pass] -r --no-parent  --reject="index.html*"'

Lister les dossiers les plus volumineux

Celui-là, je l'utilise quand je blinde trop mon disque dur. J'ai qu'un SSD de taille modérée sur mon laptop, et ça peut aider pour faire le ménage. Cette commande liste les répertoires les plus obèses dans mon répertoire personnel (/home/thomas/). Très pratique.

alias dirsizes="du -h --max-depth=1 /home/thomas/ | sort -h"

git ultra-kikoo-graph

C'est un collègue qui me la filé, tel quel. C'est une sorte de super pretty-print pour git log --graph. Ce mec était un fada de git, et j'ai appris pas mal de trucs sur le sujet à ses côtés :

alias lg="git log --color --graph --pretty=format:'%Cred%h%Creset
                  -%C(auto)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset' --abbrev-commit --date=relative"

ou (plus propre), dans votre ~/.gitconfig :

[alias]

graph = log --color --graph --pretty=format:'%Cred%h%Creset  -%C(auto)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset' --abbrev-commit --date=relative

ix.io

Vous connaissez http://ix.io ? C'est une sorte de pastebin adapté à la ligne de commande. A l'aide de curl, ça permet de poster des documents, et l'appel HTTP retourne l'url à curler pour le récupérer. Super utile pour partager des snippets de code entre collègues (attention, cependant, ce n'est absolument pas confidentiel.)

alias internet="curl -F 'f:1=<-' ix.io"

ça permet d'écrire des trucs du style:

$ cat file.py | internet
http://ix.io/oifkl

ou encore (très pratique)

$ cat ~/.ssh/id_rsa.pub | internet
http://ix.io/pjzee

ça marche même avec de petits fichiers binaires !

$ feh `cat ~/images/selfie_nude.png | internet`
*magic shows upx*
